Build Your First Neural Network

A step-by-step guide to creating a simple neural network using Python and TensorFlow.

What is a Neural Network?

A neural network is a series of algorithms that attempt to recognize patterns in data. It mimics the human brain's structure, consisting of layers of neurons that process input data and provide outputs.

Prerequisites

Before you start, make sure the following libraries are installed:

  • Python (version 3.x)
  • Pip (Python package installer)
  • TensorFlow (Install using pip install tensorflow)
  • NumPy for numerical operations (Install using pip install numpy)

Step 1: Importing Libraries

Start by importing the necessary Python libraries for building and training the neural network.


import numpy as np
import tensorflow as tf
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Dense

Step 2: Preparing the Dataset

For this tutorial, we will use a simple dataset: the XOR problem. It's a classic problem where the goal is to classify two inputs into one of two classes.


# Input data: XOR problem (binary classification)
X = np.array([[0, 0], [0, 1], [1, 0], [1, 1]])
y = np.array([[0], [1], [1], [0]])

print(X)
print(y)

The input consists of pairs of binary values, and the output is the XOR result for each pair.

Step 3: Building the Neural Network Model

We'll build a simple neural network with one hidden layer. The network will consist of an input layer, a hidden layer with two neurons, and an output layer with one neuron.


# Building the neural network model
model = Sequential()

# Adding the hidden layer with 2 neurons and 'relu' activation
model.add(Dense(2, activation='relu', input_shape=(2,)))

# Adding the output layer with 1 neuron and 'sigmoid' activation for binary classification
model.add(Dense(1, activation='sigmoid'))

# Compiling the model
model.compile(optimizer='adam', loss='binary_crossentropy', metrics=['accuracy'])

# Displaying the model summary
model.summary()

Here, we use the adam optimizer, binary_crossentropy loss function, and accuracy as the evaluation metric.

Step 4: Training the Model

Now we will train the model using the XOR data. We'll run the training for 1000 epochs.


# Training the model
model.fit(X, y, epochs=1000, verbose=1)

The verbose=1 parameter provides updates during training, showing the model's progress over the epochs.

Step 5: Evaluating the Model

Once the model is trained, we can evaluate its performance on the training data to see how well it has learned the XOR pattern.


# Evaluating the model
loss, accuracy = model.evaluate(X, y)
print(f"Loss: {loss}")
print(f"Accuracy: {accuracy}")

If the model has learned the XOR problem well, the accuracy should be 100% (1.0).

Step 6: Making Predictions

Finally, let's use the trained model to make predictions on the XOR inputs.


# Making predictions
predictions = model.predict(X)
print("Predictions:")
print(predictions)

The output will be close to 0 or 1, as the model predicts the XOR result for each input pair.

Step 7: Visualizing the Results

To visualize the results, you can print both the input data and the model's predictions side by side.


# Displaying the inputs and predicted outputs
for i in range(len(X)):
    print(f"Input: {X[i]}, Predicted Output: {predictions[i]}")
